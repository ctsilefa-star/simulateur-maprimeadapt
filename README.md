# Simulateur d'éligibilité MaPrimeAdapt' — Vivalea

Simulateur interactif d'éligibilité à l'aide MaPrimeAdapt' pour le maintien à domicile.

## 🌐 Production

URL : https://simulateur-maprimeadapt.vercel.app

## 🛠️ Stack

- **React 18** (CDN via unpkg)
- **Babel standalone** (compilation JSX in-browser)
- **EmailJS** (envoi des leads)
- **Auto-resize iframe** (postMessage pour intégration Webflow)
